1. The goal of this project is to develop task and motion planning algorithms for missions requiring a visibility-based device with a field-of-view such as a flashlight or a camera.
2. This project is meant to run on Ubuntu 18.04 with ROS Melodic and with a Husky robot which is a UR5 mounted on a mobile platform. The UR5 has a flashlight attached to it which is our visibility-based device (generating the conic FoV).
3. The context files are found in the @claude_context directory.
4. This file (@CLAUDE.md) and the global plan file (@claude_context/global_plan.txt) should never be edited.
5. A ``map'' of the project is in @claude_context/project_map.txt. This map should contain information regarding the role, significance, and contents of every file in the project, and should be kept up-to-date and used when determining where to look for information. 
6. Detailed documentation of written code is required.
