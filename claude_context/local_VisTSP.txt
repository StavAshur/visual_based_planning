# This can be used to create a local graph from which to get a local VisTSP 
Local VisRRG (r, E, B, T, k):
1) goals ← VIT.sample_target_cfgs(T, k)
2) G ← connect(goals)
3) While ∃ i ∈ [n] : goals[i] < k
   3.1) p ← sample(B) # B is the local bounding box bounding the region we can move in
   3.2) Q' ← NN_G(p, n_neighbors)
   3.3) Q ← Extend(Q', p)
   3.4) for q ∈ Q, t ∈ T: # goals[t] < k (in red)
          if visibility(FK(q), t)
            Extend(q, VisIK(q, t))
